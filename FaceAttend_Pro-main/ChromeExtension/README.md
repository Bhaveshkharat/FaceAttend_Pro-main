## Face Attendance Chrome Popup Extension

This folder contains a Chrome extension that wraps the **web build** of the Face Attendance Expo app as a **popup extension**. It does **not** affect the Android/iOS builds.

### 1. Build the web version

From the `ExpoApp` directory:

```bash
cd ../ExpoApp
npx expo export:web
```

This creates `ExpoApp/web-build/` with `index.html`, `assets/`, and compiled JS/CSS.

### 2. Copy web build into the extension

1. Go to `ExpoApp/web-build/`.
2. Copy **all files and folders** from `web-build` into this `ChromeExtension` folder.
3. Keep `manifest.json` and `README.md` in place; overwrite everything else as needed.

The final `ChromeExtension` folder should contain:

- `manifest.json`
- `index.html`
- `assets/` (and any other JS/CSS bundles generated by Expo)
- `README.md`
- Optional icons: `icon16.png`, `icon48.png`, `icon128.png`

### 3. Load the extension in Chrome

1. Open `chrome://extensions/` in Chrome.
2. Enable **Developer mode** (top-right).
3. Click **Load unpacked**.
4. Select this `ChromeExtension` folder.
5. Pin the extension (optional), then click its icon to open the popup. It will load `index.html` from the Expo web build.

### 4. Updating after changes

Whenever you change the Expo app:

1. Rebuild web:

```bash
cd ../ExpoApp
npx expo export:web
```

2. Copy the new contents of `ExpoApp/web-build/` into this `ChromeExtension` folder (overwriting old build files).
3. In `chrome://extensions/`, click **Reload** on the extension.
